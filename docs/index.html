How to Use




Welcome to CryptoBoard !

  • 分散型アプリケーションプラットフォームであるイーサリアム(Ethereum)を使った掲示板アプリです.
  • 投稿したメッセージは半永久的にブロックチェーン(イーサリアム)上に存在し続けます.
  • ご利用には暗号通貨であるetherを少額とMetamaskが必要になります. 現在モバイルではご利用頂けません.
