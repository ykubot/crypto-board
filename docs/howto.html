How to use

1. Get Ether
  • まず暗号通貨であるEtherを販売所から購入します.
  • 国内だとbitbank, bitFlyer, Zaifなどから購入できます.
2. MetaMask Install
  • 次にGoogle Chrome用のプラグインであるMetaMaskをインストールします.
  • MetaMaskはEthereumを使った分散型アプリケーション(DApps)を簡単に使うことができるプラグインであり、現在開発されているDAppsはほとんどMetaMaskを使って動作する用に作られています.
  • インストールが完了したら、アドレスを作成し購入したEtherをウォレットに送金します.
  • これでDAppsを利用する準備が完了しました.
3. Send a message on CryptoBoard !

  • Ethereumネットワークに正しく接続できていることを確認します.

    "Your Address Info"に接続しているウォレットのアドレスと残高が表示されていれば問題ありません.

  • "Send a message"からニックネームとメッセージを入力し、送信します.

  • MetaMaskの"CONFIRM TRANSACTION"が開くので、内容に問題がなければ"SUBMIT"で進みます.

    "REJECT"から取り消すことも出来ます.

  • メッセージ投稿のトランザクションが発行されます.

    発行されたトランザクションは"Block hash"のIDから確認することが出来ます.

  • トランザクションIDのリンクを開くと、Ethereum上での承認状況が見れます.

  • トランザクションが成功すると、ステータスが"Success"になります.

    承認時間はネットワークが混雑していなければ1分以内には承認されます.

  • サイトを更新すると投稿されたメッセージを確認することが出来ます.

    一度投稿されたメッセージは例え本サイトが閉じられたとしても未来永劫存在し続けます.
